VALIDATION RULES
════════════════════════════════════════════════════════════════════════════

NAME
├─ Must: Only letters, spaces, hyphens, apostrophes, periods
├─ Must: 2-100 characters
├─ Cannot: Be only numbers
└─ Cannot: Have more digits than letters

ROLL NUMBER
├─ Must: Contain at least ONE digit (key requirement!)
├─ Must: 2-50 characters
├─ Must: Only letters, numbers, hyphens, underscores
├─ Cannot: Be purely alphabetic (would be a name)
└─ Cannot: Look like a name pattern with digits mixed in

EMAIL
├─ Must: Valid format (user@domain.extension)
├─ Optional: Can be empty string
└─ If provided: Must be properly formatted

CLASS NAME
├─ Must: 1-50 characters
└─ Can: Any content
